⚙️ GHL Webhook Integration Guide — Seller Lead Page

1
Create GHL Webhook Trigger
In GHL → Automation → New Workflow → Trigger: Inbound Webhook. Copy the webhook URL provided.
2
Paste Webhook URL
In the JS below, replace YOUR_GHL_WEBHOOK_URL with the URL from Step 1. The form payload includes: firstName, lastName, phone, email, address, beds, baths, sqft, condition, situation, timeline, notes, source, refNum, submittedAt.
3
Map GHL Fields
In the workflow, map: firstName → First Name · lastName → Last Name · phone → Phone · email → Email · address → Address. Add custom fields for condition, situation, timeline.
4
Set Pipeline Stage
Add action: Add to Pipeline → Pipeline: Seller Leads → Stage: New Lead. Add tag: website-seller-lead.
5
Trigger Automation
Add action: Send SMS → "Hi [firstName], this is Ironwood Homes and Renovations. We received your info for [address] and will be in touch within 24 hours. Reference: [refNum]." Send Email confirmation. Set 24hr follow-up if no response.
6
Ad Tracking (UTM)
Append UTM params to your ad destination URL: ?utm_source=facebook&utm_campaign=seller-leads. The JS captures these automatically and includes utm_source, utm_campaign, utm_medium in the webhook payload for attribution.
7
Photo Handling
Photos are stored in browser localStorage as base64 previews for the client view. For production, replace the photo upload handler with a call to a file storage endpoint (AWS S3, Cloudinary, or GHL's media library via API) and include the file URLs in the webhook payload.
